Pokedex

Pokedex allows users to view a list of Pokemon and see more details for each one. Additionally, users can Search the Pokemon list by name and filter the list by type. The application is built with HTML, CSS, and JavaScript and loads data from an external API.

MyFlix API

The myFlix API is a RESTful backend service designed for the myFlix web application, providing comprehensive movie information including details on movies, directors, and genres. Built with Node.js, Express, and MongoDB, it supports user registration, profile management, and the ability to curate a list of favorite movies, ensuring secure and efficient data handling.

MyFlix Client: React

MyFlix is a client side app used to view a list of movies. Users can login and view information about movies, such as actors, directors, genre and more. The user can also add and remove movies from their favorites list, update their profile information and unregister. The front end is built using React and accesses the MyFlix API.

Meet App
