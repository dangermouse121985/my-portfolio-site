Pokedex

Project Description

Pokedex is a web application built with HTML, CSS, and JavaScript that allows users to browse a list of Pokemon and view detailed information about each one. The application loads data from an external API, the Pokemon API, and provides a user-friendly interface for exploring the Pokemon universe. Users can search for Pokemon by name and filter the list by type. The key features include loading data from the API, displaying a list of Pokemon with names and images, and presenting detailed information in a modal when a Pokemon is clicked.

Developing the Pokedex application enhanced my skills in working with APIs, handling asynchronous operations, and managing dynamic data. I improved my ability to create responsive layouts using Bootstrap and learned to implement complex UI patterns, such as modals, to display detailed information. Additionally, I gained experience in optimizing user experience with features like infinite scroll for loading data and implementing search and filter functionalities. The project also emphasized the importance of code quality, as I utilized ESLint for linting and Prettier for code formatting, ensuring a clean and maintainable codebase.

Project Technology

  • HTML: Structured the content of the application, creating elements such as lists, modals, and buttons.
  • CSS and Bootstrap: Styled the application using Bootstrap and custom CSS to ensure a responsive and visually appealing design.
  • JavaScript: The serverless architecture of the application is powered by AWS Lambda. This allows for scalable and cost-effective execution of server-side code without the need to manage traditional server infrastructure.
  • Pokemon API: The Pokemon API provided the data for the application
