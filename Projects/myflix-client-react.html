MyFlix Client: React

Project Description

MyFlix is a client-side application that allows users to explore a catalog of movies. Users can log in to access detailed information about movies, including actors, directors, and genres. Additionally, they can curate a personalized list of favorite movies, manage their profile, and even deregister from the app if desired. The project showcases my proficiency in building dynamic, responsive single-page applications using modern web technologies.

Throughout the development of the MyFlix application, I gained a deep understanding of component-based architecture and modern JavaScript by utilizing React and ES2015+. I enhanced my skills in state management by leveraging React Redux to handle complex state interactions, particularly for movie filtering. Additionally, I improved my UI design and responsiveness capabilities using Bootstrap to ensure the application is user-friendly and adaptable to various screen sizes. Implementing state routing enabled seamless navigation between different views, greatly enhancing the user experience. Furthermore, integrating and handling data from the MyFlix API allowed me to develop proficiency in API integration, ensuring that the app's views were populated with accurate and relevant data.

Project Technology

By incorporating the following technologies, the Meet app not only provides a robust user interface but also offers seamless integration with external services, efficient serverless functionality, and thorough testing and monitoring for a reliable and high-performance user experience.

  • React: The web application is built using React, a popular JavaScript library for building user interfaces. React's component-based architecture allowed for the creation of modular and reusable UI elements.
  • Redux Redux facilitated the implementation of the movie filtering functionality. When a user performs a search or applies filters to the movie list, Redux stores the filtered results in the state, ensuring that the UI reflects the user's selections accurately.
  • Bootstrap: Bootstrap served as the UI library for styling and responsiveness in the MyFlix application. It's grid system and responsive utility classes were used to ensure that the application's UI remained visually appealing and functional across various devices and screen sizes.
  • Parcel: Parcel served as the build tool for the MyFlix application, handling tasks such as bundling, transpiling, and code optimization.
