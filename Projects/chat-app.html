Meet App

Project Description

The React Native Chat App is a mobile application designed to facilitate communication between users. It allows users to send text messages, share images, and share their current location. Built with React Native and Expo, and using Google Firestore Database, the app ensures smooth communication with online and offline capabilities. Users can customize their chat experience by entering their name and choosing a background color for the chat screen.

Through this project, I gained in-depth knowledge of React Native and its ecosystem, including Expo for development and testing. I enhanced my understanding of integrating Firebase for authentication and Firestore for real-time data storage. Implementing the Gifted Chat library taught me how to create rich, interactive chat interfaces. I also learned how to handle media in mobile apps by enabling image picking from the gallery and capturing images with the camera. Additionally, I improved my skills in managing app state and ensuring data persistence both online and offline.

Project Technology

  • React Native: React Native was used to develop the mobile application: It's cross platform support allowed me to develop a single codebase that runs on both iOS and Android.
  • Expo: Provided the tools to streamline the development process, including a simulator for testing on multiple devices.
  • Google Firestore Database: Firestore was used for real-time data storage and synchronization. It stored chat messages, images, and location data.
  • Google Firebase Authentication: Implemented anonymous authentication for user access control.
  • Gifted Chat Library: Used to create teh chat interface, providing a rich set of customizable components for chat bubble, input fields, and more.
